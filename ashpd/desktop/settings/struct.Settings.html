Struct ashpd::desktop::settings::Settings

source · 
pub struct Settings<'a>(/* private fields */);
Expand description

The interface provides read-only access to a small number of host settings required for toolkits similar to XSettings. It is not for general purpose settings.

Wrapper of the DBus interface: org.freedesktop.portal.Settings.

pub async fn receive_setting_changed_with_args<T>( &self, namespace: &str, key: &str ) -> Result<impl Stream<Item = Result<T, Error>>, Error>where T: TryFrom<OwnedValue>, Error: From<<T as TryFrom<OwnedValue>>::Error>,

Similar to Self::receive_setting_changed but allows you to filter specific settings.

Example
use ashpd::desktop::settings::{ColorScheme, Settings};
use futures_util::StreamExt;

let settings = Settings::new().await?;
while let Some(Ok(scheme)) = settings
    .receive_setting_changed_with_args::<ColorScheme>(
        "org.freedesktop.appearance",
        "color-scheme",
    )
    .await?
    .next()
    .await
{
    println!("{:#?}", scheme);
}
